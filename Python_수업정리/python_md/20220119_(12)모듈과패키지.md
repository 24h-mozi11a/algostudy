## 모듈과 패키지 

### 모듈(Module)

- 모듈(module)은 특정 기능을 하는 코드를 파이썬 파일(.py) 단위로 작성한 것

### 패키지

- 특정 기능과 관련된 여러 모듈의 집합 
- 패키지 안에는 또 다른 서브 패키지를 포함



#### 모듈과 패키지 불러오기 

``` python
import module 
from module import var, function, Class
from module import * 

from package import module
from package.module import var, function, Class
```

``` python
import random
```
